You have an application that runs at a customer's site. Sometimes, the application crashes
because of heap corruption. You ask the customer to enable full page heap on the application
process so that you can troubleshoot the heap corruption.
The customer sends you a user dump file. You need to identify if the full page heap was enabled
when the user dump was created. Which WinDbg command should you use?
A. !gflag
B. !heap
C. !verifier
D. !vm
