Explanation: * Pros of a Terminal Server only implementation
Following are the benefits of a Terminal Server only implementation:
No need to upgrade any workstations: This may be a great approach for companies where
many client computers do not meet the requirements for Dynamics GP and there are no
other reasons to upgrade.
Installation, updates, maintenance, and support are all simplified tremendously: For

incremental amount of work and cost to set up the rest of the users on the same Terminal
