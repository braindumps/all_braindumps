provided by the service must be transmitted using a WSDL XML format over HTTP.
B. A Web Service is a software component that is described in a machine-processable
format. Other systems interact with the Web Service in a manner prescribed by the

HTTP with an XML serialization.

packets in response to requests from other systems. The requests must be in XML format,
