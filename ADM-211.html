Describe the difference between a lookup and a master-detail relationship?
Answer: Lookup: Creates a relationship that links this object to another object. The
relationship field allows users to click on a lookup icon to select a value from a popup list.
The other object is the source of the values in the list Master-detail: Creates a special type
of parent-child relationship between this object (the child, or "detail") and another object
(the parent, or "master").
Describe the difference between an email template and a mail merge template:

Describe the differences between Case and an If function?
Answer: Case Checks a given expression against a series of values. If the expression is
equal to a value, returns the corresponding result. If it is not equal to any values, it returns
the else _result
IF Determines if expressions are true or false.
Returns a given value if true and another value if false
