Your network contains a server that runs Windows Server 2008 R2. You plan to create a
custom script.
You need to ensure that each time the script runs, an entry is added to the Application
event log.
Which tool should you use?
A. Eventcreate
B. Eventvwr
C. Wecutil
D. Wevtutil
Answer: A

useful as a diagnostic tool in scripts when you record an error or event directly into the logs
without using VBScript or another language to log the event.
