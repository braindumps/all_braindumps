A system administrator has added a server cluster as a member of a service integration
bus (SIB). The administrator needs to configure one messaging engine for each server in
this cluster and to have a failover location for each messaging engine.
What should the administrator do to achieve the required design?
A. Configure a new messaging engine in each server to support failover.
B. Configure the data store for each messaging engine for each server in the cluster.
C. Select scalability with high availability as the messaging engine policy.
D. Create a core group policy to restrict each messaging engine to a particular server.
Answer: C
Explanation: The scalability with high availability configuration ensures that there is a
messaging engine for each server in a cluster, and that each messaging engine has a
failover location.
