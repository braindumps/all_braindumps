groups that perform at different levels. LUN A is located in the disk group with the highest
performance level. Cloud Service B is hosted by Virtual Server B. Virtual Servers A and B
are hosted by HypervisorA, which is installed on a physical server (not shown) that resides

and resides in Cloud B, which is located in a different geographic region than Cloud A.

an automated scaling listener responsible for initiating scaling activities.
