You are responsible for the configuration of MegaCorp’s Check Point Firewall. You need to
allow two NAT rules to match a connection. Is it possible? Give the BEST answer.
A. Yes, it is possible to have two NAT rules which match a connection, but only when using
Automatic NAT (bidirectional NAT).
B. Yes, it is possible to have two NAT rules which match a connection, but only in using
Manual NAT (bidirectional NAT).
C. Yes, there are always as many active NAT rules as there are connections.
D. No, it is not possible to have more than one NAT rule matching a connection. When the
firewall receives a packet belonging to a connection, it compares it against the first rule in
the Rule Base, then the second rule, and so on. When it finds a rule that matches, it stops
checking and applies that rule.
Answer: A
